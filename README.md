# Visualizador de Cierre Convexo para Discos

Esta es una implementación en Java y JavaFX del algoritmo de geometría computacional propuesto en el siguiente artículo:

> **Referencia Principal:**
> D. Rappaport, "A convex hull algorithm for discs, and applications," *Computational Geometry: Theory and Applications*, vol. 1, pp. 171-187, 1992.
